# Closur — Conventions

## Code
- TypeScript throughout
- Cloudflare Agents SDK patterns (Agent class, @callable decorators, useAgentChat hook)
- Vite for build
- Tailwind for styling
- No unnecessary abstractions — hackathon pace, ship fast

## Design
- Dark theme: #0a0a0f bg, #b4e62e green accent
- Fonts: Fira Code (mono) + Space Grotesk (sans)
- Corner bracket panel decorations
- See docs/brand-guidelines.md for full brand system

## Naming
- Product: Closur (closur.ai)
- Tagline: "The life you want is on the other side of a few hard conversations."

## Git
- No Claude mentions in commits
- Concise commit messages
- dev branch as working branch, push to main for deploys
